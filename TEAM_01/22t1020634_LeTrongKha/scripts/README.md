## auto_push.bat
Script tự động để push code lên GitHub repository.

### Cách sử dụng:
1. Double-click vào file `auto_push.bat`
2. Script sẽ tự động:
   - Clone repository mới từ GitHub
   - Copy files đã cập nhật
   - Commit và push lên GitHub

### Lưu ý:
- Đảm bảo Git đã được cài đặt
- Cần có quyền truy cập vào repository
- Script chỉ ảnh hưởng đến folder cá nhân, không ảnh hưởng các team khác
